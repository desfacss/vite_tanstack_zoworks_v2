## ğŸ”§ To Enable Real AI

### 1. Environment Variables
Create/update `.env`:
```env
VITE_ENABLE_AI=true
VITE_GOOGLE_AI_API_KEY=your_gemini_key_here
VITE_AI_MODEL=gemini-1.5-pro
```

### 2. Add Route
Update your router configuration:
```tsx
import AIWorkbench from '@/modules/ai/pages/AIWorkbench';

// Add route:
{
  path: '/ai/workbench',
  element: <AIWorkbench />,
}
```

### 3. Implement AI Backend
Update `aiService.ts`:
```tsx
import { streamText } from 'ai';
import { google } from '@ai-sdk/google';

export async function streamAIResponse(messages, options) {
    const result = await streamText({
        model: google(import.meta.env.VITE_AI_MODEL),
        messages,
    });
    
    for await (const chunk of result.textStream) {
        options?.onChunk?.(chunk);
    }
    options?.onComplete?.();
}
```
